Start LED blinking.

Colors can be combinated using the OR operator.

Start blinking the LED(s) according the requested time. To stop blinking use ledz_on, ledz_off, ledz_toggle or ledz_set functions. This function can also be used to stop the blinking passing zero to time_on or time_off.

Create ledz object.

When creating a ledz the type must be passed as the first argument and it must be one of the values defined by ledz_type_t enumeration. The second argument must be an array containing the color(s) of the LED being created. The third argument must be an integer array of the port and pin of each LED in correspondence with the previous color(s).

Examples:

// create one color LED
ledz_create(LEDZ_1COLOR, (const ledz_color_t []){LEDZ_RED}, (const int []){0, 1});
// create RGB LED
const ledz_color_t colors[] = {LEDZ_RED, LEDZ_GREEN, LEDZ_BLUE};
const int pins[] = {0, 1, 0, 2, 0, 3};
ledz_create(LEDZ_3COLOR, colors, pins);
Parameters
[in]typemust one of the values in ledz_type_t declaration
[in]colorsits a ledz_color_t type array containing the LED colors
[in]pinsan integer array of the port and pin of each LED
Returns
pointer to stimer object or NULL if no more timers are available

Fade in LED brightness.

Colors can be combinated using the OR operator.

Progressively turn on the LED using the given rate. The rate is determined in milliseconds per brightness unit. e.g.: rate = 10 means that the brightness is increased by one every 10ms. The brightness will increase until max value.

Parameters
[in]ledledz object pointer
[in]colorthe color to adjust
[in]ratefade in rate to turn on the led
[in]maxthe maximum brightness value (stop value)

◆ ledz_tick()

void ledz_tick ( void  )

The tick function.

This function must be used to define the clock of the ledz core. It must be called from an interrupt service routine (ISR). The period of the interruption must be set using the LEDZ_TICK_PERIOD macro.
